Click on this interface - http://audio-ai.ahlab.org:8000/?app=algo1

  • Using the 'Select Model' dropdown, load the model called 'Hits & Scratches'.
  • Using the 'Select Example' dropdown, Load the example called 'Hit - Wood-Dirt'.
  • This sound is created by a drum stick hitting a wooden tree trunk.
  • Change the controls provided on the left panel to update the generated 'Synthetic Sound'. Observe how the 'Reconstructed Sound' changes with the 'Synthetic Sound'.
  • Using the controls on the left panel, change this example of 'Hit - Wood-Dirt' into a sound made by a drum stick scratching a soft surface (such as a carpet or sofa).
